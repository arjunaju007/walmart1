to delete the category (http://localhost:8080/api/categories/{id}

For customer (http://localhost:8080/api/customers)

1) to save it http://localhost:8080/api/customers

2) to get customer by id same for the update the customer: http://localhost:8080/api/customers/{id}

3) to logged in and validate : http://localhost:8080/api/customers/validate

-------------------------------------------------------------------------------

for product (http://localhost:8080/api/products):


1) to save and retrive (http://localhost:8080/api/products)

2) to find the product by id (http://localhost:8080/api/products/{id})

3) to find the product by category (http://localhost:8080/api/products/{cats})

4) to update the product by id (http://localhost:8080/api/products/{id})

5) to delete the product by id (http://localhost:8080/api/products/{id})
